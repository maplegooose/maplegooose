So I began creating this website a few weeks ago with a bunch of testing and fails. I tried to create my first website purely in html, which went terribly when I tried applying a Jekyll theme to it. It was honestly really difficult finding out what to do.

With enough time, I was able to learn how to set up a basic website and run it on Github. Honestly, I was really proud because people can finally view the website I had created.

The next step was to chooose a better theme I liked. The themes on Github were cool, but I felt like I could do better. I browsed through several new themes of Github and discovered [Mr. Green](https://github.com/MrGreensWorkshop/MrGreen-JekyllTheme). It's a really beautiful theme, and I love it, but I ran into a huge issue.

This website was a multilingual website, which would allow me to have my website in different languages. As of now, I don't plan for that, so I tried to remove the Japanese language only to run into several new problems. From there, I was also really proud of learning how to remove the Japanese side of the website since I did it all by myself.

In general, I feel like I'm getting a hang of coding for the most part now that I'm actually taking initiative in creating new projects. After I rebrand this website and turn it into something more personal, I plan on approaching two different projects. The first project I want to create is a simple rock paper scissors game on a seperate page. I want to get used to doing actual coding on a webpage so I can create more projects for school in the future. I plan on creating many calculators for myself, and maybe my friends, to use for school. The second project is to create a website for my school's orchestra, since the themes and layouts on Google Sites kind of suck.

Overall, I'm pretty proud of my progress. This feels like something I can actively dedicate my time and effort into, and it feels so exciting just seeing all the progress. Making mistakes and failing doesn't disuade me at all, I actually feel giddy about the progress itself rather than the result. I hope that I can create something truly more amazing in the future. Anyways, I'm now signing out.
